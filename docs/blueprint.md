# **App Name**: LeaveEase

## Core Features:

- Leave Request Form: Submission form with fields for leave type (Paternity, Annual, Sick, etc.), start date, and end date.
- Automated Leave Calculation: Automated calculation of leave days, excluding Sundays and public holidays.
- Automated Workflow Routing: Routing of leave request form to employee's supervisor upon submission.
- Supervisor Approval Interface: Approval/rejection interface for supervisors, with space for providing a reason.
- Manager Approval Interface: Second-level approval/rejection interface for the general manager, after supervisor approval.
- Admin Configuration Panel: Administrator space to configure employee accounts and information such as contract type, start date, end date, and supervisor.
- AI-Powered Approver Suggestions: Suggest approvers in the chain of approvals using AI, if the user puts some vague role title for an approver instead of an actual person. The AI tool can search employee roles and generate options. Use the employee title, team membership, etc. to generate its list of suggestions.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke a sense of calm and reliability, important for managing leave requests.
- Background color: Light gray (#F0F4F8) to ensure a clean and professional look.
- Accent color: Teal (#26A69A) to draw attention to important actions and status indicators.
- Font pairing: 'Inter' (sans-serif) for headlines and body text; known for clarity and readability in digital interfaces.
- Use simple, clear icons from a set like Material Design Icons to represent different types of leave and actions (approval, rejection, etc.).
- Maintain a clean and well-spaced layout with clear visual hierarchy to facilitate easy navigation and form completion.
- Subtle transitions and animations on button clicks and status updates to provide visual feedback without being distracting.
